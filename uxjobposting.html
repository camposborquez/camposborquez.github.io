Solicitamos: UX WEB/MOBILE DESIGNER

Objetivo: Crear productos que brinden experiencias significativas y relevantes a los usuarios en el uso de un software. Esto implica el diseño de todo el proceso de adquisición e integración del producto, incluidos los aspectos de marca, diseño, usabilidad y función.

Requisitos:

  • Licenciatura/Ingeniería en Informática, diseño gráfico o carreras a fin
  • Experiencia mínima de 2 años en diseño y desarrollo de interfaces web/móviles demostrable con portafolio
  • Manejo de Adobe Design Suite, Photoshop u otros
  • Dominio de HTML, CSS y JavaScript
  • Capacidad para crear interfaces responsivas en diferentes dispositivos
  • Desarrollo Frontend con tecnologías como Angular, React, Vue
  • Conocimientos en peticiones HTTP
  • Deseable manejar preprocesadores CSS como SCSS, SASS, LESS

Responsabilidades:

  • Crear mockups, flujos y/o prototipos que comuniquen con precisión la funcionalidad de un producto
  • Desarrollo de interfaces interactivas y atractivas para el usuario asegurando una buena presentación de un producto
  • Recopilar y analizar datos que lleven a una toma de decisiones en cuanto a la experiencia del usuario en un producto
  • Diseñar iconos e imágenes ilustrativas necesarias para la presentación de los productos

Competencias:

  • Mentalidad centrada en el usuario, atención al detalle
  • Excelente comunicación verbal y escrita
  • Capacidad de decisión para asegurar la mejor manera de ofrecer una grata experiencia de usuario
  • Facilidad para poder transmitir un mensaje a través de imágenes, colores, posiciones.

Ofrecemos:

  • Buen ambiente de Trabajo
